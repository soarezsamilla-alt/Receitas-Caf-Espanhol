# **App Name**: Nutri Café

## Core Features:

- Recipe Library & Display: Showcase a catalog of +200 breakfast recipes, detailing preparation time, calorie count, and ingredients. Features structured sections for easy browsing.
- Bonus Content Access: Provide dedicated sections to highlight and display exclusive bonus content like sugar-free desserts, meal prep guides, and detox juice recipes.
- Nutritionist Profile: Present an 'About the Author' section with details about the nutritionist Camila Moura, building credibility and authority.
- Testimonial Carousel: Implement a scrolling carousel to display user testimonials and social proof in an engaging visual format.
- Interactive FAQ: An accordion-style Frequently Asked Questions section to address common user queries in a user-friendly and expandable manner.
- Purchase Call-to-Action: Strategically placed buttons throughout the page to direct users to the product purchase page (e.g., https://pay.kiwify.com.br/SEU-LINK-AQUI).
- Personalized Recipe AI Tool: A generative AI tool that suggests recipe variations or new breakfast ideas based on user input, such as available ingredients, preferred preparation time, or specific dietary goals (e.g., high protein, low carb), focusing on quick, tasty, and healthy options for weight loss.

## Style Guidelines:

- Primary color: Energetic Orange (#E8600A) for headings and key interactive elements, reflecting vitality and appetite. (HSL: H:25, S:90%, L:48%)
- Background color: Soft Cream (#FDF3E7), providing a warm, clean, and inviting base for all content. (HSL: H:30, S:86%, L:95%)
- Analogous accent: Subtle Coral Red (#E4666B) for delicate highlights or secondary design elements, harmonizing with the primary orange. (HSL: H:355, S:70%, L:65%)
- Functional accent: Vibrant Green (#2CAC43) for primary call-to-action buttons, creating strong visual contrast and prompting action.
- Body font: 'Roboto', a versatile sans-serif font for readability and clarity. Note: currently only Google Fonts are supported.
- Headline font: 'Sora', a modern sans-serif font, used for prominent titles to convey a contemporary and bold feel. Note: currently only Google Fonts are supported.
- Combination of simple symbolic icons (like checkmarks) and expressive, food-related emojis to list benefits, maintaining a friendly yet informative tone.
- A well-structured long-form landing page layout with clearly defined sections for product presentation, benefits, testimonials, and FAQ. Utilizes a responsive grid for content blocks and centered content with max-width constraints for optimal readability on all devices.
- Subtle animations enhance interactivity: continuous scroll for the hero image carousel, smooth transitions for FAQ accordion, and gentle hover effects (e.g., slight lift and shadow change) on buttons and cards.
